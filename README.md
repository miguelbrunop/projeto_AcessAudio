# AcessAudio

Protótipo que converte texto em áudio utilizando a biblioteca gTTS (Google Text-to-Speech).

## Objetivo
O projeto tem como finalidade apoiar estudantes no aprendizado e promover acessibilidade para pessoas com deficiência visual, permitindo que textos escritos sejam transformados em arquivos de áudio.

## Como funciona
1. O usuário define um texto em português.
2. O programa converte o texto em fala.
3. O áudio é salvo em um arquivo `.mp3`.

## Estrutura do projeto
- `main.py` → código principal
- `saida.mp3` → exemplo de áudio gerado
